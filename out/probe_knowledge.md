以下是为给定医学表格特征生成的临床先验知识，针对结局变量 `hospital_expire_flag`（住院死亡标志）。所有内容基于常见临床指南和研究证据，以 Markdown 表格形式呈现。

| 特征名 | 单变量强度（摘要） | 临床关联描述 | 建议阈值 | 证据置信度（高/中/低） |
| --- | --- | --- | --- | --- |
| Respiratory_failure | 强 | 呼吸衰竭常需机械通气，导致氧合障碍和通气不足，显著增加多器官功能衰竭及住院死亡风险 | 存在（诊断为呼吸衰竭） | 高 |
| SOFA | 强 | 序贯器官衰竭评估评分，每升高1分死亡风险增加约10%–20%，≥10分时死亡率极高（>50%） | ≥10 | 高 |
| resp_rate | 中等 | 呼吸频率异常（过快提示代偿或呼吸窘迫，过慢提示中枢抑制）与不良预后相关，是早期预警指标 | >24次/分 或 <8次/分 | 中 |
| sepsis | 强 | 脓毒症导致全身炎症反应和组织灌注不足，是ICU患者最常见的死亡原因之一 | 存在（符合Sepsis-3诊断标准） | 高 |
| Kidney_failure | 强 | 急性肾衰竭常与多器官功能不全相关，需肾脏替代治疗，显著增加住院死亡率 | 存在（诊断为急性肾损伤/肾衰竭） | 高 |
| Urea_Nitrogen | 中等 | 血尿素氮升高反映肾功能下降或高分解代谢状态，与死亡率呈正相关，尤其用于评估危重患者预后 | >20 mg/dL | 中 |
| heart_rate | 中等 | 持续心动过速（>100次/分）增加心肌耗氧和风险，心动过缓（<50次/分）提示传导异常或休克，均与死亡相关 | >100次/分 或 <50次/分 | 中 |
| Malignancy | 强 | 活动性恶性肿瘤患者基础状态差，免疫功能低下，化疗/放疗加重器官负担，住院死亡风险显著升高 | 存在（活动性恶性肿瘤） | 高 |
| age | 强 | 高龄（>70岁）患者生理储备减少，合并症增多，对急性应激反应能力下降，死亡率呈指数增长 | >70岁 | 高 |
| Solidtumors | 中 | 实体肿瘤患者死亡风险升高，但受肿瘤类型、分期、治疗状态影响，预测强度低于活动性恶性肿瘤 | 存在（确诊实体瘤） | 中 |
| inr | 中 | INR升高提示凝血功能障碍（肝病、DIC、抗凝药物），增加出血风险及器官损伤，与死亡率独立相关 | >1.5 | 高 |
| temperature | 中等 | 高热（>38.5℃）提示感染/炎症，低体温（<36℃）常为脓毒症休克或代谢衰竭表现，两者均与死亡风险增加相关 | >38.5℃ 或 <36℃ | 中 |
| ICH | 强 | 颅内出血导致颅内高压、脑疝及神经功能损害，即使在ICU中死亡率也极高（>30%–50%） | 存在（诊断为ICH） | 高 |
| Magnesium | 低 | 镁离子异常（低镁常见于营养不良/利尿剂使用；高镁见于肾衰）可能诱发心律失常，但作为独立死亡预测因子证据有限 | <1.6 mg/dL 或 >2.4 mg/dL | 低 |
| Chloride | 弱 | 氯离子异常多继发于酸碱失衡或肾功能障碍，单独预测死亡的临床意义不明确，需结合其他电解质评估 | 无明确阈值（临床意义不明确，常与其他电解质联合评估） | 低 |
| bicarbonate | 中等 | 碳酸氢盐降低提示代谢性酸中毒（常见于休克、肾衰、乳酸酸中毒），酸中毒程度与死亡率密切相关 | <20 mEq/L | 中 |
| Cerebral_infarction | 中 | 脑梗死导致神经功能缺损及并发症（吸入性肺炎、深静脉血栓），死亡风险增加但低于脑出血 | 存在（诊断为脑梗死） | 中 |
| ARDS | 强 | 急性呼吸窘迫综合征引起顽固性低氧血症和肺顺应性下降，机械通气时间长，死亡率高达30%–40% | 存在（符合柏林定义，如PaO2/FiO2 ≤300 mmHg） | 高 |
| wbc | 中等 | 白细胞显著升高（>12×10⁹/L）提示感染/应激；降低（<4×10⁹/L）提示免疫抑制或骨髓衰竭，均与不良预后相关 | >12×10⁹/L 或 <4×10⁹/L | 中 |
| Hepatic_failure | 强 | 肝衰竭导致凝血障碍、黄疸、肝性脑病及多器官衰竭，未行肝移植者死亡率极高（>50%） | 存在（诊断为肝衰竭） | 高 |
