| Respiratory_failure | 强 | 急性呼吸衰竭需机械通气，直接反映心肺功能严重受损，是院内死亡的强烈独立危险因子。 | 存在（是/否） | 高 |
| age | 中 | 年龄增长伴随生理储备下降、共病增多，住院死亡风险随年龄连续上升，高龄患者更易出现不良结局。 | 无明确阈值，风险随年龄递增；临床常以≥65岁为老年分界，但并非突变点 | 高 |
| sepsis | 强 | 脓毒症是感染引发的全身性炎症反应和器官功能障碍，可快速进展为脓毒性休克和多器官衰竭，是ICU死亡主因。 | 存在（根据Sepsis-3标准） | 高 |
| wbc | 中 | 白细胞极度升高或降低均提示严重感染、炎症或骨髓抑制，形成“U”型风险曲线，与不良预后相关。 | <4×10⁹/L 或 >12×10⁹/L | 中 |
| SOFA | 强 | SOFA（序贯器官衰竭评估）综合反映呼吸、循环、肝肾、凝血等多系统功能，评分越高说明器官衰竭程度越重，死亡风险急剧增加。 | ≥2分（存在器官功能障碍）；通常≥2分即示警，动态升高更危险 | 高 |
| Cerebral_infarction | 中 | 脑梗死可导致意识障碍、误吸、瘫痪和脑水肿，急性期或大面积梗死显著增加院内死亡风险。 | 存在（是/否） | 高 |
| gender | 弱 | 性别对院内死亡的单变量影响较弱且不一致，部分研究显示女性相对保护，但受混杂因素影响大，不建议作为独立预测规则。 | 无明确阈值（二分类变量，无单变量阈值） | 低 |
| hemoglobin | 中 | 重度贫血降低氧输送，加重心脏负荷；显著升高的血红蛋白增加血液黏滞度和血栓风险，均与院内死亡呈U型关联。 | <10 g/dL 或 >16 g/dL | 中 |
| mbp | 强 | 平均动脉压（MBP）是器官灌注压的核心指标，<65 mmHg 持续存在提示组织低灌注和休克，与死亡率显著相关。 | <65 mmHg | 高 |
| ICH | 强 | 颅内出血（脑出血）引起颅内压急剧升高、脑疝和神经功能恶化，起病急、死亡率极高。 | 存在（是/否） | 高 |
| Sodium | 中 | 低钠血症和高钠血症均反映机体水电解质平衡严重紊乱，与神经功能障碍、细胞水肿/脱水及死亡风险U型关联。 | <135 mmol/L 或 >145 mmol/L | 高 |
| Magnesium | 中 | 低镁与心律失常、癫痫样发作相关；高镁多因摄入过多或肾衰竭，两者均可增加危重患者死亡风险，但独立作用强度弱于钠。 | <1.5 mg/dL（0.6 mmol/L）或 >2.5 mg/dL（1.0 mmol/L） | 中 |
| dbp | 强 | 舒张压（DBP）过低反映血管张力丧失或心输出量不足，影响冠脉灌注，尤其<60 mmHg时与心血管事件和死亡密切关联。 | <60 mmHg | 高 |
| Urea_Nitrogen | 强 | 血尿素氮升高提示肾前性氮质血症、急性肾损伤或慢性肾病，是重症患者不良预后的稳健预测因子。 | >20 mg/dL（7.14 mmol/L） | 高 |
| Chloride | 中 | 显著高氯或低氯通常与酸碱失衡、肾功能异常相关，高氯血症可加重酸中毒，增加死亡风险，但关联弱于钠异常。 | <98 mmol/L 或 >107 mmol/L | 中 |
| caclium | 中 | 离子钙或校正钙异常（低钙多见于脓毒症、急性胰腺炎；高钙与恶性疾病、内分泌危象相关），均与危重患者死亡率U型曲线相关。 | 校正钙 <8.5 mg/dL（2.12 mmol/L） 或 >10.5 mg/dL（2.62 mmol/L） | 中 |
| inr | 强 | INR升高反映凝血功能障碍，常见于肝衰竭、DIC或过度抗凝，预示出血风险显著增高，与院内死亡强烈关联。 | >1.5 | 高 |
| Diabetes | 中 | 糖尿病患者住院期间易并发感染、高/低血糖事件及心肾并发症，增加死亡风险，但单变量贡献受年龄和合并症影响。 | 存在（是/否） | 高 |
| bicarbonate | 强 | 碳酸氢根显著下降提示代谢性酸中毒（如乳酸酸中毒、酮症酸中毒），是组织低灌注的敏感指标，与死亡风险紧密相关。 | <22 mmol/L | 高 |
| resp_rate | 强 | 呼吸频率是生命体征中最敏感的恶化信号，显著增快（>20）或减慢（<12）均提示严重心肺或代谢紊乱，>30预示危重结局。 | >20 breaths/min 或 <12 breaths/min（危重警戒 >30） | 高 |
