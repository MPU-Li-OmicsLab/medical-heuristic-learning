以下是针对住院死亡（hospital_expire_flag）的临床先验知识表格。该表格基于危重症医学文献和临床经验，为每个特征提供了单变量关联强度、临床解释、常用阈值及证据置信度。

| 特征名 | 单变量强度（摘要） | 临床关联描述 | 建议阈值 | 证据置信度（高/中/低） |
| :--- | :--- | :--- | :--- | :--- |
| Respiratory_failure | 强（OR > 5，死亡率升高3-5倍） | 呼吸衰竭导致严重低氧/高碳酸血症，常需机械通气，直接增加呼吸肌疲劳和多器官衰竭风险 | 存在（诊断编码或PaO₂<60 mmHg / PaCO₂>50 mmHg） | 高 |
| SOFA | 强（每增加1分，死亡风险升高10-20%；AUC≈0.75-0.85） | 序贯器官衰竭评估的综合评分，反映心、肺、肝、肾、凝血、神经功能，总分越高，死亡风险呈指数上升 | SOFA ≥ 7（中高危）；SOFA ≥ 10（极高危） | 高 |
| resp_rate | 中等（U形关系，>24或<8次/分风险上升，OR≈2-3） | 呼吸过快提示呼吸窘迫或代谢代偿，过慢提示中枢抑制或呼吸肌衰竭，均可预示病情恶化 | > 24 次/分 或 < 8 次/分 | 中 |
| sepsis | 强（脓毒症患者死亡率约20-30%，脓毒性休克>40%） | 全身感染导致炎症风暴、血管麻痹、组织低灌注，是ICU最常见死亡原因之一 | 存在（sepsis-3诊断标准或临床诊断） | 高 |
| Kidney_failure | 强（AKI需RRT者死亡率增加2-4倍） | 急性肾衰竭导致尿毒症毒素蓄积、液体超负荷、电解质紊乱，显著加重其他器官损伤 | 存在（KDIGO 2-3期或需要肾脏替代治疗） | 高 |
| Urea_Nitrogen | 中等（升高10mg/dL对应死亡风险增加20-30%） | 血尿素氮受肾前、肾性、分解代谢影响，持续升高提示肾灌注不足或高分解状态，常见于危重病晚期 | > 20 mg/dL（轻度升高）；> 40 mg/dL（显著升高） | 中 |
| heart_rate | 中等（持续>100次/分或<50次/分，OR≈1.5-2.5） | 心率过快增加心肌氧耗，过缓提示传导异常或心输出量不足，快速房颤等心律失常尤需警惕 | > 100 次/分 或 < 50 次/分（持续30分钟以上） | 中 |
| Malignancy | 中-高（活动性肿瘤患者死亡风险增加1.5-3倍） | 恶性肿瘤本身导致免疫抑制、营养不良、血栓风险；血液肿瘤伴粒细胞缺乏感染后死亡率极高 | 存在（活动期，不包括既往治愈） | 高 |
| age | 强（每增加10岁，死亡风险约升高1.3-1.5倍） | 高龄患者器官储备下降、合并症多、对治疗耐受性差，是独立且最强预测因子之一 | > 65 岁（高风险）；> 80 岁（极高风险） | 高 |
| Solidtumors | 中等（实体瘤转移或晚期OR≈1.5-2） | 实体肿瘤晚期可压迫重要脏器、导致恶液质、增加感染风险，但整体风险略低于血液肿瘤 | 存在（转移或未控制） | 中 |
| inr | 中等（INR>1.5时出血风险增加，>2.0死亡风险翻倍） | 凝血功能障碍提示肝功能受损、DIC或抗凝药物影响，INR升高与颅内出血、多器官衰竭相关 | > 1.5（异常）；> 2.0（显著异常） | 中 |
| temperature | 中等（U形关系，低体温<36°C风险高于高热） | 低体温（<36°C）常见于脓毒症休克早期或代谢崩溃，高热（>39°C）提示严重感染或中枢损伤 | < 36°C 或 > 39°C（核心体温） | 中 |
| ICH | 强（院内死亡率30-50%，血肿量>30ml风险剧增） | 颅内出血直接损伤脑组织、引起占位效应和颅内高压，常需手术干预，预后极差 | 存在（CT确诊，且血肿量>15ml） | 高 |
| Magnesium | 低-中（极端水平OR≈1.5，但研究异质性大） | 低镁（<1.7 mg/dL）可致心律失常、心肌收缩力下降；高镁（>2.5 mg/dL）可致神经肌肉抑制，但作为独立预测因子证据不充分 | < 1.7 mg/dL 或 > 2.5 mg/dL | 低 |
| Chloride | 低（高氯>110 mmol/L时OR≈1.2-1.5，缺乏强证据） | 高氯血症常伴高氯性酸中毒，可能与急性肾损伤有关，但多数研究未证实其独立预测价值 | > 110 mmol/L（高氯）；< 95 mmol/L（低氯） | 低 |
| bicarbonate | 中等（HCO₃⁻<22 mmol/L时OR≈2-3） | 降低提示代谢性酸中毒（如乳酸酸中毒、肾衰竭），是病情严重程度的重要标志，升高可见于代谢性碱中毒 | < 22 mmol/L（酸中毒） | 中 |
| Cerebral_infarction | 中-高（大面积梗死死亡率20-40%） | 急性脑梗死导致神经功能缺损、脑水肿、吸入性肺炎等并发症，体积大或累及关键区域风险高 | 存在（影像学确诊，且体积≥1/3大脑中动脉供血区） | 高 |
| ARDS | 强（轻度ARDS死亡率27%，重度约45%） | 急性呼吸窘迫综合征表现为顽固性低氧、肺泡渗出，易并发多器官功能衰竭，需机械通气及肺保护策略 | 存在（柏林标准，PaO₂/FiO₂≤300 mmHg） | 高 |
| wbc | 中等（WBC>15k或<4k，OR≈1.5-2.5） | 白细胞显著升高提示感染或应激反应，极度降低提示骨髓抑制或败血症，两者均与不良结局相关 | > 15,000 /μL 或 < 4,000 /μL | 中 |
| Hepatic_failure | 强（急性肝衰竭死亡率>50%，伴肝性脑病更高） | 肝脏衰竭导致凝血障碍、肝性脑病、肝肾综合征、感染，病情进展迅速，死亡率极高 | 存在（胆红素>5 mg/dL，INR>1.5，伴肝性脑病） | 高 |

**说明：**  
- 所有阈值均为临床常用参考值，实际应用中需结合患者具体情况（如年龄、基础病）进行动态评估。  
- 对于二分类特征（如呼吸衰竭、脓毒症），“存在”即视为阳性，无需额外连续阈值。  
- 置信度基于多项大型研究、meta分析或临床指南的一致性；低置信度特征（如镁、氯）提示变量可能作为辅助参考，不宜单独用于决策。
