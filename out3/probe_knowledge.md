以下是为您提供的医学表格特征生成的临床先验知识，基于结局变量 `hospital_expire_flag`（住院死亡标志）。表格包含每个特征的单变量强度摘要、具体临床关联描述、建议阈值以及证据置信度。

| 特征名 | 单变量强度（摘要） | 临床关联描述 | 建议阈值 | 证据置信度 |
|--------|-------------------|--------------|----------|------------|
| Respiratory_failure | 强正向关联，存在呼吸衰竭患者死亡风险升高3~5倍 | 呼吸衰竭导致低氧血症或高碳酸血症，直接危及生命，常需机械通气，显著增加死亡风险 | 存在该诊断即视为高风险 | 高 |
| SOFA | 强正向关联，每增加1分死亡风险升高约20%~30% | SOFA评分反映多器官功能障碍严重程度，分值越高，器官衰竭个数越多，住院死亡率急剧上升 | ≥2分提示器官功能障碍，≥10分死亡率极高（建议用于风险分层） | 高 |
| resp_rate | U型关联，过高或过低均增加死亡风险 | 呼吸频率>24次/分提示呼吸代偿或窘迫，<8次/分提示呼吸抑制，均与不良结局相关 | >24次/分或<8次/分 | 高 |
| sepsis | 强正向关联，存在脓毒症患者死亡风险升高2~4倍 | 脓毒症是感染引起的全身炎症反应，可导致休克、多器官衰竭，是ICU常见死亡原因 | 存在该诊断即视为高风险 | 高 |
| Kidney_failure | 强正向关联，存在肾衰竭患者死亡风险升高2~3倍 | 急性肾损伤或慢性肾衰竭急性加重导致代谢废物蓄积、电解质紊乱，增加心脏负荷和感染风险 | 存在该诊断即视为高风险 | 高 |
| Urea_Nitrogen | 正向关联，每升高10 mg/dL死亡风险增加约15%~20% | 尿素氮升高反映肾前性或肾性肾功能不全，也提示高分解代谢状态，与重症患者预后不良相关 | >20 mg/dL（肾功能不全）；>50 mg/dL（严重） | 中（受多种因素影响，如脱水、出血） |
| heart_rate | U型关联，过快或过慢均增加死亡风险 | 心率>100次/分常提示应激、感染或心功能不全；<50次/分可能为心脏传导阻滞或药物作用 | >100次/分或<50次/分 | 高 |
| Malignancy | 正向关联，存在恶性肿瘤患者死亡风险升高1.5~2倍 | 恶性肿瘤本身及化疗、放疗等降低免疫功能，增加感染和器官衰竭风险，尤其晚期患者预后差 | 存在该诊断即视为高风险 | 高 |
| age | 强正向关联，每增加10岁死亡风险升高约1.5~2倍 | 高龄患者器官储备功能下降，合并症多，对重症打击的耐受性差，死亡率随年龄增加显著上升 | >65岁（老年）；>80岁（极高风险） | 高 |
| Solidtumors | 正向关联，存在实体肿瘤患者死亡风险升高1.5~2倍 | 与恶性肿瘤类似，实体肿瘤可导致局部压迫、远处转移、恶液质，影响治疗结局 | 存在该诊断即视为高风险 | 中（与肿瘤分期及活动性相关） |
| inr | 正向关联，每升高0.5单位死亡风险增加约20%~30% | INR升高提示凝血功能障碍，常见于肝病、DIC或使用抗凝药物，增加出血风险和器官功能障碍 | >1.5（轻中度异常）；>2.0（重度，出血风险高） | 高 |
| temperature | 双向关联，过高或过低均增加死亡风险 | 高热（>39°C）提示严重感染或中枢性发热；低体温（<35°C）提示休克或代谢衰竭，均与不良预后相关 | >39°C或<35°C | 中（受退热药物、测量部位影响） |
| ICH | 强正向关联，存在颅内出血患者死亡风险升高3~6倍 | 颅内出血导致颅内压升高、脑疝形成，常需手术或重症监护，死亡率极高 | 存在该诊断即视为高风险 | 高 |
| Magnesium | 双向关联，过高或过低均增加死亡风险 | 低镁血症（<1.7 mg/dL）可致心律失常、神经肌肉兴奋；高镁血症（>2.6 mg/dL）可致呼吸抑制、心脏传导阻滞 | <1.7 mg/dL 或 >2.6 mg/dL | 中（受肾功能、药物影响较大） |
| Chloride | 双向关联，过高或过低均增加死亡风险 | 高氯血症（>110 mmol/L）常见于脱水或代谢性酸中毒；低氯血症（<95 mmol/L）常见于呕吐、利尿剂使用，均可加重酸碱失衡 | >110 mmol/L 或 <95 mmol/L | 中（需结合钠离子及酸碱状态） |
| bicarbonate | 负向关联（低值），每降低5 mmol/L死亡风险增加约20%~40% | 碳酸氢盐降低提示代谢性酸中毒（如乳酸酸中毒、肾衰竭），反映组织低灌注或酸碱失衡严重 | <20 mmol/L（轻度酸中毒）；<15 mmol/L（重度） | 高 |
| Cerebral_infarction | 正向关联，存在脑梗死患者死亡风险升高1.5~2.5倍 | 大面积脑梗死可导致脑水肿、颅内高压、肺部感染等并发症，增加住院死亡率 | 存在该诊断即视为高风险 | 中（与梗死面积、位置、溶栓时机相关） |
| ARDS | 强正向关联，存在ARDS患者死亡风险升高3~5倍 | 急性呼吸窘迫综合征导致严重低氧血症，常需机械通气，且易并发多器官衰竭 | 存在该诊断即视为高风险 | 高 |
| wbc | 双向关联，过高或过低均增加死亡风险 | 白细胞显著升高（>20×10^9/L）提示严重感染或炎症；白细胞减少（<4×10^9/L）提示免疫抑制或骨髓抑制 | >20×10^9/L 或 <4×10^9/L | 中（受应激、药物、血液系统疾病干扰） |
| Hepatic_failure | 强正向关联，存在肝衰竭患者死亡风险升高4~8倍 | 肝衰竭导致凝血障碍、黄疸、肝性脑病、代谢紊乱，死亡率极高，常需肝移植支持 | 存在该诊断即视为高风险 | 高 |

**说明**：  
- 单变量强度摘要基于流行病学及重症医学文献中的常见效应估计。  
- 建议阈值主要参考临床实践指南（如Sepsis-3、KDIGO、SOFA评分标准）和常用实验室参考范围。  
- 证据置信度根据该指标与死亡率关联的一致性、生物合理性及临床研究数量综合判断。  
- 对于二分类诊断特征（如呼吸衰竭、脓毒症），存在即视为高风险，因此阈值设置为“存在该诊断”。
